What is a Data Type?

In programming, a data type describes the kind of value a variable can hold. For example, it could be a number, a piece of text, or a list of items.

Python automatically detects the data type based on the value you assign to a variable. This is called dynamic typing.

Getting the Data Type

You can get the data type of any object by using the type() function:

Python
x = 5
print(type(x))    # <class 'int'>

y = "Hello"
print(type(y))    # <class 'str'>

z = 3.14
print(type(z))    # <class 'float'>

1. Text Type

str (String)

  • Represents text.
  • Created by putting characters inside single quotes (' ') or double quotes (" ").
  • Strings are immutable, meaning they cannot be changed after they are created.
Python
name = "Alice"
message = 'Hello, Python!'

2. Numeric Types

int (Integer)

  • Whole numbers, positive or negative, no decimal point.
  • Unlimited length (only limited by memory).
Python
a = 5
b = -10
c = 1000000

# Operations
a + b
a * 10
a ** 2   # exponent (5 squared = 25)

float

Numbers with a decimal point. Used for measurements and calculations with fractions.

Python
pi = 3.14159
temp = -2.5

complex

Numbers with a real and an imaginary part (represented by j). Used in scientific and engineering calculations.

Python
z = 3 + 5j

3. Sequence Types

Used to store multiple values in a specific order.

list

  • Ordered, changeable, and allows duplicates.
  • Written with square brackets [ ].
Python
fruits = ["apple", "banana", "cherry"]

tuple

  • Ordered, unchangeable, and allows duplicates.
  • Written with round brackets ( ).
Python
coordinates = (10, 20)

range

Represents a sequence of numbers, often used in loops.

Python
nums = range(5)  # 0, 1, 2, 3, 4

4. Mapping Type

dict (Dictionary)

  • Stores data in key-value pairs.
  • Unordered (since Python 3.7, ordered), changeable, and no duplicate keys.
  • Written with curly brackets { }.
Python
person = {"name": "John", "age": 25}

5. Set Types

Used to store unique items.

set

  • Unordered and unindexed. No duplicate items.
  • Written with curly brackets { }.
Python
unique_nums = {1, 2, 3, 3}  # Result: {1, 2, 3}

frozenset

Like a set, but unchangeable (immutable).

6. Boolean Type

bool

  • Represents one of two values: True or False.
  • Used in logic and decisions (if statements).
Python
is_loading = True
is_finished = False

8. None Type

NoneType

  • Represents the absence of a value.
  • Used to define a null variable or return value.
Python
x = None
i

Setting the Specific Data Type: You can convert between types using functions like int(), str(), float(), list(), etc. This is called casting.
