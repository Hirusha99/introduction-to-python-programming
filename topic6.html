Lesson 6: Python Match Statement

Introduced in Python 3.10, the match statement provides a more readable and efficient way to handle multiple conditions — similar to switch in C++ or Java.

45 minLab 06Week 06

Syntax

The match statement compares a variable against several possible values (patterns) and executes the block of code for the first matching pattern.

Python
match variable:
    case pattern1:
        # code for pattern1
    case pattern2:
        # code for pattern2
    case _:
        # code if no patterns match (default case)
i

The _ (underscore) is a wildcard pattern that acts as a default case — it catches everything that doesn't match any other pattern.

Example

Python
status_code = 404
match status_code:
    case 200:
        print("Success")
    case 404:
        print("Not Found")
    case 500:
        print("Server Error")
    case _:
        print("Unknown Status")

Default Value

The case _: block acts as a default when no other case matches.

Python
day = "Wednesday"
match day:
    case "Monday":
        print("Start of work week")
    case "Friday":
        print("End of work week")
    case "Saturday" | "Sunday":
        print("Weekend!")
    case _:
        print("Midweek day")

Combine Values

You can combine multiple values in one case using the | (or) operator:

Python
color = "red"
match color:
    case "red" | "orange":
        print("Warm color")
    case "blue" | "green":
        print("Cool color")
    case _:
        print("Other color")

Why Use Match?

  • Readability: It is cleaner than using many elif statements.
  • Powerful Pattern Matching: It can match complex data structures (like lists or dictionaries).
  • Better Structure: It clearly defines alternatives for a specific variable.

Comparing If-Else vs. Match

Python — If-Else approach
if status_code == 200:
    print("Success")
elif status_code == 404:
    print("Not Found")
else:
    print("Unknown Status")
Python — Match approach
match status_code:
    case 200:
        print("Success")
    case 404:
        print("Not Found")
    case _:
        print("Unknown Status")
