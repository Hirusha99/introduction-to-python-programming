Lesson 4: Python Operators

Operators are used to perform operations on variables and values. Python divides operators into 7 groups.

55 minLab 03Week 04

1. Arithmetic Operators

Arithmetic operators are used with numeric values to perform common mathematical operations:

OperatorNameExample
+Additionx + y
-Subtractionx - y
*Multiplicationx * y
/Divisionx / y
%Modulus (remainder)x % y
**Exponentiation (power)x ** y
//Floor division (integer)x // y

Division in Python

Python has two division operators:

Python
x = 12
y = 5

# / Division — always returns a float
print(x / y)   # Result: 2.4

# // Floor division — always returns an integer (rounds DOWN)
print(x // y)  # Result: 2

2. Assignment Operators

Assignment operators are used to assign values to variables:

Python
x = 5       # Assign
x += 3      # Same as x = x + 3
x -= 3      # Same as x = x - 3
x *= 3      # Same as x = x * 3
x /= 3      # Same as x = x / 3
x %= 3      # Same as x = x % 3
x //= 3     # Same as x = x // 3
x **= 3     # Same as x = x ** 3

3. Comparison Operators

Comparison operators are used to compare two values. They return True or False:

OperatorNameExample
==Equalx == y
!=Not equalx != y
>Greater thanx > y
<Less thanx < y
>=Greater than or equal tox >= y
<=Less than or equal tox <= y

4. Logical Operators

Logical operators are used to combine conditional statements:

Python
x = 5
# and — Returns True if both statements are true
print(x < 10 and x > 2)    # True

# or — Returns True if one of the statements is true
print(x < 3 or x > 2)      # True

# not — Reverse the result
print(not(x < 5 and x > 0))  # False

5. Identity Operators

Identity operators are used to compare the objects — not if they are equal, but if they are actually the same object in memory:

Python
x = ["apple", "banana"]
y = ["apple", "banana"]
z = x

print(x is z)      # True  — same object
print(x is y)      # False — same values, different objects
print(x is not y)  # True

6. Membership Operators

Membership operators are used to test if a sequence is present in an object:

Python
fruits = ["apple", "banana", "cherry"]

print("apple" in fruits)      # True
print("mango" not in fruits)  # True

7. Bitwise Operators

Bitwise operators are used to compare binary numbers:

OperatorNameDescription
&ANDSets each bit to 1 if both bits are 1
|ORSets each bit to 1 if one of two bits is 1
^XORSets each bit to 1 if only one of two bits is 1
~NOTInverts all the bits
<<Left shiftShift left by pushing zeros in from the right
>>Right shiftShift right by pushing copies of the leftmost bit
